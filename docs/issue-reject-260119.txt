◆ 주요 기능

【실시간 타이머】
• 월드보스, 지옥물결, 군단 이벤트 카운트다운
• 현재 진행 중인 이벤트 상태 표시
• 다음 이벤트 시작 시간 안내

【홈 화면 위젯】
• 소형/중형 위젯 지원
• 앱을 열지 않고도 홈 화면에서 바로 확인
• 깔끔하고 직관적인 디자인

【맞춤 알림】
• 이벤트별 알림 개별 설정
• 1분, 5분, 10분, 15분, 30분 전 알림 선택
• 원하는 이벤트만 알림 받기

【편리한 사용성】
• 오프라인 완벽 지원 - 인터넷 연결 불필요
• 라이트/다크 모드 및 시스템 테마 지원
• 심플하고 빠른 인터페이스

◆ 이런 분께 추천합니다

• 바쁜 일상 속에서도 월드보스를 놓치고 싶지 않은 분
• 지옥물결 시작 시간을 매번 확인하기 번거로운 분
• 위젯으로 간편하게 이벤트 일정을 관리하고 싶은 분

◆ 앱 정보

• 비공식 팬메이드 앱입니다
• 게임사와 무관하며 공식 앱이 아닙니다
• 무료로 제공되며 인앱 구매가 없습니다
• 무료로 제공되며 광고도 없습니다.
