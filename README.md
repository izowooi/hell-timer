## 프로젝트 소개

Hell Timer는 **Diablo 4의 주요 월드 이벤트 시간을 추적하는 모바일 앱**입니다.

서버 없이 100% 로컬에서 이벤트 시간을 계산하며, 지옥물결(Helltide), 군단(Legion), 월드보스(World Boss) 세 가지 이벤트의 다음 시작 시간을 실시간으로 알려줍니다.

---

## 주요 기능

| 기능 | 설명 |
|:----:|:-----|
| **실시간 타이머** | 세 가지 이벤트의 남은 시간 및 다음 시작 시간 표시 |
| **홈 화면 위젯** | 앱을 열지 않고도 이벤트 시간 확인 |
| **푸시 알림** | 이벤트 시작 전 사용자 정의 시간에 알림 |
| **오프라인 동작** | 네트워크 연결 없이 완전한 기능 |
| **글로벌 동기화** | UTC 기반으로 전 세계 동일한 시간 표시 |
| **다국어 지원** | 한국어, 영어 |

## 이벤트 시간 규칙

Hell Timer가 추적하는 세 가지 Diablo 4 월드 이벤트입니다.

### 이벤트 비교

| 이벤트 | 주기 | 계산 방식 | 특징 |
|:------:|:----:|:---------:|:----:|
| **지옥물결** | 60분 | 로컬 시간 | 매시 정각 시작, 55분 활성 |
| **군단** | 25분 | UTC 앵커 | 전 세계 동기화 |
| **월드보스** | 105분 | UTC 앵커 | 하루 약 14회 |

---

### 지옥물결 (Helltide)

가장 예측하기 쉬운 이벤트입니다.

```
┌─────────────────────────────────────────────────────────────┐
│  00:00                                    54:59   55:00  59:59
│    ├──────────── 활성 (55분) ──────────────┤├── 휴식 (5분) ──┤
└─────────────────────────────────────────────────────────────┘
```

- **활성**: 매시 00분 ~ 54분 (55분간)
- **휴식**: 매시 55분 ~ 59분 (5분간)

---

### 군단 (Legion)

25분 고정 주기로 발생하며, 글로벌 동기화됩니다.

| 항목 | 값 |
|:----:|:--:|
| 주기 | 25분 (1,500초) |
| 앵커 | 1,200초 (UTC 기준점) |
| 활성 시간 | 약 4분 |

**계산 공식:**
```
다음 이벤트 = 앵커 + ⌈(현재 - 앵커) / 주기⌉ × 주기
```

---

### 월드보스 (World Boss)

105분(1시간 45분) 주기로 스폰됩니다.

| 항목 | 값 |
|:----:|:--:|
| 주기 | 105분 (6,300초) |
| 앵커 | 1,767,702,600 (UTC 기준점) |
| 일일 횟수 | 약 13-14회 |

### 핵심 설계 원칙

1. **서버리스** - 모든 이벤트 시간을 로컬에서 계산
2. **UTC 기반 동기화** - 전 세계 사용자에게 동일한 시간 표시
3. **독립적 위젯** - 메인 앱 없이도 위젯이 자체 계산 수행
4. **네이티브 구현** - 각 플랫폼의 최신 UI 프레임워크 활용
