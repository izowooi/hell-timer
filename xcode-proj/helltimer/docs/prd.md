디아블로4 의 월드보스, 지옥물결, 군단 이벤트의 알림을 사용자에게 직관적으로 알려줄 수 있는 앱을 만들고 싶습니다.
모던한 swift UI 를 사용해서 직관적인 UI 를 구성해야합니다.
위젯을 제공하여 사용자가 앱을 실행하지 않아도, 월드보스, 지옥물결, 군단 이벤트가 얼마나 남았는 지 알 수 있어야합니다.
알림은 사용자가 3가지를 모두 등록해서 받아볼 수 있으며, 만일 하나도 등록하지 않았다면 별도의 Notification 은 오지 않아야합니다.
테스트 케이스를 만들어서 테스트를 모두 통과해야합니다.
코드는 SOLID 원칙에 따라 작성되어야합니다.
비행기 모드에서도 동작할 수 있어야합니다.
아래의 2개의 api 는 각각 diablo4.life 에 있는 지옥물결, 월드보스의 api 입니다.

https://diablo4.life/api/trackers/helltide/reportHistory
https://diablo4.life/api/trackers/worldBoss/reportHistory

또한 아래와 같이 월드보스를 제외한 나머지 둘은 정확히 예측이 가능합니다.
이 둘은 로컬에서 계산한 정보를 기반으로 해주세요.
월드보스는 인터넷 데이터를 우선하되, fallback 으로 로컬에서 계산한 결과를 반영해주세요.
지옥물결계산 기반 (매시 정각, 55분 지속) - 100% 예측
군단계산 기반 (25분 주기) - 앵커 타임만 있으면 예측 ( 앵커타임 1월2일 금요일 20:35, 21:00 ... )
월드보스API 필요 (보스 종류/위치가 변동)
